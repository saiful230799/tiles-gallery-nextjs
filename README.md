# 🏺 Tiles Gallery - Modern Web Application

Welcome to **Tiles Gallery**, a premium web platform for discovering and managing exquisite tile collections. Built with the latest tech stack, this application offers a seamless user experience, responsive design, and robust functionality. 🇲🇾

## 🚀 Live Demo
Check out the live application here: [Tiles Gallery Live](https://tiles-gallery-pied.vercel.app)

---

## ✨ Features

*   **Public Exploration**: Anyone can browse the latest tiles directly from the homepage and view detailed information without logging in.
*   **Dynamic Search & Filter**: Easily find tiles by name or sort them by price (Low to High / High to Low).
*   **Secure Authentication**: Integrated with **Better Auth** for secure login and registration.
*   **User Dashboard**: Logged-in users can manage their own profile and view specific collections.
*   **CRUD Operations**: Users can add new tiles, update details, and manage their listings.
*   **Animated UI**: Smooth Marquee animations and modern card designs using **Tailwind CSS** and **DaisyUI**.

---

## 🛠️ Tech Stack

*   **Frontend**: Next.js 14+ (App Router), React
*   **Styling**: Tailwind CSS, DaisyUI
*   **Database**: MongoDB (Atlas)
*   **Authentication**: Better Auth
*   **Deployment**: Vercel

## 📦 Getting Started

Follow these steps to run the project locally:

1.  **Clone the repository**:
    ```bash
    git clone [https://github.com/saiful2331/tiles-gallery.git](https://github.com/saiful2331/tiles-gallery.git)
    cd tiles-gallery
    ```

2.  **Install dependencies**:
    ```bash
    npm install
    ```

3.  **Set up Environment Variables**:
    Create a `.env.local` file in the root directory and add the following:
    ```env
    MONGODB_URI=your_mongodb_connection_string
    BETTER_AUTH_SECRET=your_secret
    BETTER_AUTH_URL=http://localhost:3000
    ```

4.  **Run the development server**:
    ```bash
    npm run dev
    ```
    Open [http://localhost:3000](http://localhost:3000) in your browser.
